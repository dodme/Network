LAN(Local Area Network)
 - 근거리 통신망
 - 범위가 넓지않은 한정적인 공간에서 연결되있는 통신망
 - 내가 초대하고싶은 사람들만 추려서 사용할수 있는 작은범위의 네트워크

WAN(Wide Area Network)
 - 원거리 통신망
 - LAN들이 뭉쳐져서 하나의 큰 덩어리를 이룬것
 - 인터넷서비스

Unicast
 - 1:1통신, 카카오톡, 기타 메신저 등 1:1 채팅
 - 내가 통신하고자하는 대상을 콕 집어서 하는 통신의 방식

Multicast
 - 1:N통신, 카카오톡 단톡방, 기타 메신저 1:N 채팅
 - Unicast의 확장판
 - 내가 통신하고자하는 대상'들'과 통신하는 방식

Broadcast
 - 1:all 통신, 라디오방송, 이장님 방송, 아파트 관리실방송
 - 통신을 받는사람의 의지와 상관 없이 무조건 받아야하는 통신

회선/대역폭
 - 회선 : 데이터 전송의 통로
 - 대역폭 : 전송되는 데이터를 허용할 수 있는 동시접속자 수
 - 도로 자체는 회선, 도로의 폭(몇차선인지)는 대역폭이라고 생각하면 쉬움

ISP(Internet Service Provider)
 - 인터넷 서비스 공급자
 - 다양한 회선 상품을 제공하며 기업마다 서비스가 다름
 - 쉽게말해 통신사(kt, U+, Sk등)을 말하는것이다
 - ISP는 kt, u+, sk등을 통틀어서 말하는것이고, kt, u+, sk등은 한국에서 사용하는 ISP이다

전용선
 - 회선계의 First Class, 서비스 Good, 안정성 Good, 비쌈
 - 회사에서 제공해주는 회선중 하나인데 비싸고 안정적이고 문제가 생겼을때 우선적으로 조치를 받을수 있는 장점이 있다
 
VPN(Virtual Private Network)
 - 가상사설망, ISP 즉, 제 3자에 정보를 넘겨주지 않고 익명성을 유지하여 인터넷에 접속

DSL(Digital Subscriber Line)
 - 디지털가입자회선, A(Asymmetric)DSL, V(Very high-data rate)DSL
 - A(비대칭)
  ㄴ 전화선에 모뎀과 마이크로 필터를 사용해서 인터넷에 연결, 전화국으로부터 거리가 멀어도 OK
     비대칭적으로 다운로드속도는 빠르지만 업로드속도는 느리다는 특징이 있다
     지역에 따라 인적이 드문 시골등에 광케이블이 아닌 전신주(전봇대)를 통해 통신서비스 공급이 가능하기때문에 아직까지도 일부 이용함
 - V(초고속)
  ㄴ 대칭/비대칭 모두 지원하며 ADSL처럼 전화선을 이용하나 속도가 훨씬 빠름, 전화국으로부터 거리가 멀면 안됨

광통신(FTTH : Fiber To The Home)
 - 초고속 기가 인터넷, 집안까지 광케이블을 통해 인터넷을 제공하는 서비스
 - FTTH가 ADSL보다 약 100배 빠름

IP(Internet Protocol) Address
 - 컴퓨터 네트워크에서 기기들이 서로를 인식하고 통신하기 위해 사용하는 식별번호
 - 각각의 집마다 주소가 있듯이 사용하는 네트워크를 사용하려는 IT장비들에게 부여되어있는 네트워크 주소

URL
 - IP주소는 숫자로 되어있는데 이 주소를 우리가 이해하기 쉬운 주소로 바꿔놓은것

공인IP/사설IP
 - 공인IP : External / Public IP
 - 사설IP : Internal / Privacy IP
 - 공유기를 거치기 전에는 공인IP, 공유기를 거친 후에는 사설IP
 - 내 의도에 따라 IP를 제배분한것이 사설IP

케이블(Cable)
 - 컴퓨터와 네트워크 장비들 사이에서 전기적 신호를 운반하는 전송매체
 - UTP(Unshielded Twisted Pair) Cable = Lans 선 = 인터넷선
 - UTP케이블에는 CAT(카테고리)라는 규격을 가지고있다
 - UTP케이블에 CAt숫자의 의미는 회선에 최대 올릴수 있는 데이터의 양(대역폭)을 의미한다

모뎀(Modem)
 - MOdulator and DEModulator / 아날로그 <-> 디지털
 - 컴퓨터같은 디지털기기에서 아날로그 신호를 이용하여 디지털 데이터 통신을 하게 해 줌
 - 통신을 하기 위해선 전기신호(아날로그)를 다른지역으로 보내줘야하는데 컴퓨터는 디지털신호만 알아듣기때문에 이 신호를 변환해주는 역할

공유기(Home Router)
 - ISP에서 할당해주는 하나의 IP를 이용하여 여려 대의 기기가 인터넷에 접속할 수 있도록 하는 기기
 - ISP업체에서 받아온 공인IP를 여려개의 사설IP로 바꿔주는게 핵심역할
 - 파란색 LAN Port에서는 공인IP를 받아오는 역할을 하고 노란색 LAN Port에서는 공인IP를 사설IP로 변환해서 할당하는 역할을 한다
 - 유/무선공유기는 사설IP를 공유해주는 방식을 유선과 무선을 둘 다 지원하는것이다
